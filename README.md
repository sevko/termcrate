`termcrate` is a terminal port of the popular Super Crate Box.

###Installation
`make termcrate`

###Usage
`make run`

###Controls
    w - jump
    a - move left 
    d - move right
    j - shoot
    F1 - quit
    

###Instructions
Avoid the enemies, and shoot them if possible.
